About The Product

ACE aims to combat the lack of efficiency in commercial parking garages. This system scans the license plate and make of the car to determine if it is allowed into the garage. Because each license plate is associated with a specific user, administrators will know which registered vehicles are parked in the garage at any time. Through an android mobile app, ACE provides accessibility for both users and admins. Users must register their vehicle before entering a parking garage, and the admin has to approve the request in order for the vehicle to be granted access in the future. When the user approaches the garage, they are permitted entry if there is a match for that vehicle in the database. If an unregistered vehicle enters the garage, an alarm will sound.
